[ENG]

RandomStuff is not a single modification, but a set of individual options that can be installed on top of other mods, either individually or together.

List of additions:
- New affixes
- New gadgets
- Random minibosses
- Additional cabins and baskets
- Colosseum battles
- Traders caravans
- Additional difficulty levels
- Quick target acquisition with missile launchers
- Klop (sml1) available in Glukhoe

This modification is also compatible with other mods based on the original game.

Commod 3.0.6 or higher is required to install this modification.
